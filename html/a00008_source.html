1 # VitalityX Interrupt 3Fh (63)
2 
3 Interrupt 3Fh is the standard VitalityX software interrupt. You interrupt 3Fh to run VitalityX functions while in usermode.
4 
5 For example, to write to Bochs (emulation software) output (in Intel syntax):
6 ```
7  mov eax,0x01 ; Set mode to Bochs Output
8  mov ebx,string ; Set ebx to string pointer
9  int 3fh ; Interrupt VitalityX
10 string: db 'Hello VitalityX!',0
11 ```
