Detailed Description

This is a VXBFS File.

It stores the 11 character name, a GUID, an ext, flags, next modblock, nextfile and a md boolean. The GUID is a unique identifier so multiple VXBFS files can share the same name. The ext is the VXBFS File type. The md means "More Data", this signifies that there should be a search for more data after this. The flags are data which is for the OS.
