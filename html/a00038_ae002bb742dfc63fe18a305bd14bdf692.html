void* liballoc_alloc ( int  )

This is the hook into the local system which allocates pages.

It accepts an integer parameter which is the number of pages required. The page size was set up in the liballoc_init function.

Returns
NULL if the pages were not allocated.
A pointer to the allocated memory.
